Pengenalan Class & Object

Perhatikan kode berikut:

Car.java
public class Car {
    // Blueprint (Class)
    String color;
    int speed;

    public static void main(String[] args) {
        // Object Creation
        Car myCar = new Car();
        
        myCar.color = "Red";
        myCar.speed = 100;
        
        System.out.println("Mobil saya berwarna " + myCar.color);
    }
}
Analogy: Blueprint vs Car
lightbulb

Konsep Dasar

Bayangkan 'Class' sebagai cetak biru (blueprint) di pabrik. Kamu tidak bisa mengendarai cetak biru itu.

'Object' adalah mobil asli yang sudah jadi dari cetak biru tersebut. Kita bisa membuat ribuan mobil (objects) dari satu cetak biru (class).
